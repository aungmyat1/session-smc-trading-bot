## Status legend

| Status | Meaning |
|---|---|
| ACTIVE | Has a current production/import/deployment/test-suite path. This means repository-active, not proof that a process is running now. |
| PARTIALLY USED | Runnable or test-covered, but not wired into a current non-test runtime/deployment path. |
| UNUSED | Archived or has no detected launch/import/test/deployment path. |
| BROKEN | Cannot parse or has a repository-confirmed blocking defect on its intended path. |
| UNKNOWN | Evidence is insufficient, typically package markers or externally invoked surfaces. |
