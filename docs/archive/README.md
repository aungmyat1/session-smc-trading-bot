# Archived Documentation Index

This folder indexes documents intentionally preserved for historical reference. Do not treat these as authoritative for current operations unless explicitly referenced by an owner.

Locations
- `docs/Archive/` — archived strategy-specific and historical evidence
- `archive/` — older archive collections and prototype reports

Recommended usage
- Search here for historical decisions, past verdicts, and legacy reports.
- For active guidance and runbooks, prefer documents under `docs/` top-level directories (e.g., `docs/operations/`, `docs/architecture/`, `docs/svos/`).

If you intend to promote any archived document back to active status, create a PR that updates its header with `Owner`, `Last reviewed`, and `Status` fields and add a line in `docs/documentation_inventory.md` tracking the change.
